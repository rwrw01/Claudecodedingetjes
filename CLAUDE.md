# Project Rules

## Wiskunde hoofdstukken

### Voorbeelden
- Pas altijd de voorbeelden aan ten opzichte van de gegeven voorbeelden: kies net andere letters en getallen zodat het eigen materiaal is.

### Interactiviteit
- Maak alle uitleg zo interactief mogelijk: sliders, invulveldjes per tussenstap, stap-voor-stap onthulling.
- Niet alleen een antwoordveld aan het einde, maar ook tussentijdse invulvelden (bijv. "wat is de teller na vereenvoudigen?").

### Colofon
- Elke pagina heeft een footer-colofon: gemaakt door Ralph Wagter, met Claude Code, vrij hergebruik, link naar GitHub repo.

### Licentie
- De repo gebruikt de EUPL-1.2 licentie.
