1. Wat is een wortelvergelijking?

Een wortelvergelijking is een vergelijking waarbij de onbekende x onder een wortelteken staat, zoals √x of √(3x + 1).

Voorbeelden: √x = 4  |  3 + √(2x) = 7  |  √(x+1) + √x = 5
💡 Tip: Wortels zijn alleen gedefinieerd voor niet-negatieve getallen. Dus het domein is altijd: de uitdrukking onder het wortelteken ≥ 0.

Domeinbepaling

Voordat je oplost, bepaal je het domein: voor welke waarden van x is de vergelijking überhaupt zinvol?

Voorbeeld: √(4x − 3) = 5
Domein: 4x − 3 ≥ 0 → x ≥ ¾

🔍 Probeer zelf: Wat is het domein van √(5x − 10) ?

Stel de uitdrukking onder het wortelteken ≥ 0:

5x − 10 ≥ 0 → x ≥

2. Strategie: Isoleren en kwadreren

De standaardmethode om wortelvergelijkingen op te lossen:

Stap 1: Isoleer het wortelteken (zet √ alleen aan één kant)
Stap 2: Kwadrateer beide kanten (√a)² = a
Stap 3: Los de ontstane vergelijking op
Stap 4: Controleer je oplossingen (vreemde oplossingen!)

🎚️ Ontdek: (√x)² = x voor x ≥ 0

Beweeg de slider en zie wat er gebeurt als je kwadrateert:

x = 9 → √9 = 3 → (√9)² = 9 ✓

Stap-voor-stap voorbeeld

Los op: 2√x + 3 = 11

Stap 1: Isoleer √x
2√x = 11 − 3 = 8 → √x = 4
Stap 2: Kwadrateer beide kanten
(√x)² = 4² → x = 16
Stap 3: Controleer
2√16 + 3 = 2·4 + 3 = 8 + 3 = 11 ✓
Stap 0/3
⚠️ Let op: Je mag alleen kwadreren als het wortelteken al geïsoleerd is. Vergelijking √x + 2 = 5 eerst omschrijven naar √x = 3, daarna pas kwadreren!

3. Controleren op vreemde oplossingen

Het kwadreren kan vreemde oplossingen introduceren die niet in de originele vergelijking kloppen. Altijd terugvullen!

Voorbeeld met vreemde oplossing:
√(3x + 4) = x − 2
Kwadreer: 3x + 4 = (x−2)² = x² − 4x + 4
→ x² − 7x = 0 → x(x − 7) = 0
Oplossingen: x = 0 of x = 7

Controleer x = 0: √(3·0+4) = √4 = 2, maar 0−2 = −2 ✗ (vreemde oplossing!)
Controleer x = 7: √(3·7+4) = √25 = 5, en 7−2 = 5 ✓
Enige oplossing: x = 7
💡 Waarom gebeurt dit? Als je kwadrateert, "vergeet" je of de oorspronkelijke uitdrukking positief of negatief was. √ geeft altijd de positieve uitkomst.

🔍 Controleer zelf:

Vergelijking: √(2x + 1) = x − 1. Na kwadreren vind je x = 0 of x = 4.

Vul x = 0 in: √(2·0+1) = √1 = 1. Maar x − 1 = 0 − 1 = −1.

Is x = 0 een geldige oplossing?

6. Oefensommen (20 opgaven)

Los alle vergelijkingen exact op. Gebruik als decimaalteken een komma.

Voortgang: 0 van 20 correct
Opgave 1
√x = 7
x =
  1. √x = 7 is al geïsoleerd.
  2. Kwadrateer: x = 7² = 49
  3. Controleer: √49 = 7 ✓
Kwadrateer beide kanten direct.
Opgave 2
√x − 3 = 5
x =
  1. Isoleer: √x = 5 + 3 = 8
  2. Kwadrateer: x = 8² = 64
  3. Controleer: √64 − 3 = 8 − 3 = 5 ✓
Tel 3 op bij beide kanten eerst.
Opgave 3
4√x − 2 = 10
x =
  1. Isoleer √x: 4√x = 12 → √x = 3
  2. Kwadrateer: x = 3² = 9
  3. Controleer: 4√9 − 2 = 4·3 − 2 = 10 ✓
Zet eerst √x alleen: deel door 4.
Opgave 4
5√x + 1 = 21
x =
  1. Isoleer: 5√x = 20 → √x = 4
  2. Kwadrateer: x = 16
  3. Controleer: 5·4 + 1 = 21 ✓
Trek 1 af, deel dan door 5.
Opgave 5
√(2x + 6) = 4
x =
  1. √ is al geïsoleerd.
  2. Kwadrateer: 2x + 6 = 16
  3. Los op: 2x = 10 → x = 5
  4. Controleer: √(10+6) = √16 = 4 ✓
Kwadrateer beide kanten: 2x+6 = 16.
Opgave 6
√(5x − 5) = 5
x =
  1. Kwadrateer: 5x − 5 = 25
  2. Los op: 5x = 30 → x = 6
  3. Controleer: √(30−5) = √25 = 5 ✓
Kwadrateer beide kanten: 5x−5 = 25.
Opgave 7
3 + √(x − 1) = 7
x =
  1. Isoleer: √(x−1) = 4
  2. Kwadrateer: x − 1 = 16
  3. Los op: x = 17
  4. Controleer: 3 + √16 = 3 + 4 = 7 ✓
Trek 3 af van beide kanten.
Opgave 8
6√x − 2x = 0
x =
  1. Isoleer: 6√x = 2x → 3 = √x (voor x ≠ 0)
  2. Kwadrateer: x = 9
  3. x = 0 is ook een oplossing (controleer zelf!), maar de vraag vraagt x ≠ 0 → x = 9
  4. Controleer: 6√9 − 2·9 = 18 − 18 = 0 ✓
Schrijf 2x = 2·√x·√x en deel door √x (voor x>0).
Opgave 9
√(4x + 1) − 1 = 2x

Geef de kleinste geldige oplossing

x =
  1. Isoleer: √(4x+1) = 2x + 1
  2. Kwadrateer: 4x + 1 = 4x² + 4x + 1
  3. Vereenvoudig: 0 = 4x² → x = 0
  4. Controleer x=0: √1 − 1 = 0 = 2·0 ✓
Isoleer de wortel, dan kwadreren.
Opgave 10
√(x + 5) = x − 1
x =
  1. Kwadrateer: x + 5 = x² − 2x + 1
  2. Herschrijf: x² − 3x − 4 = 0
  3. Ontbind: (x−4)(x+1) = 0 → x=4 of x=−1
  4. Controleer x=−1: √4 = 2, maar −1−1 = −2 ✗ (vreemd)
  5. Controleer x=4: √9 = 3 = 4−1 ✓ → x = 4
Let op vreemde oplossingen na het kwadreren!
Opgave 11
√(2x + 3) = x − 2
x =
  1. Kwadrateer: 2x + 3 = x² − 4x + 4
  2. Herschrijf: x² − 6x + 1 = 0
  3. abc-formule: discriminant = 36 − 4 = 32
  4. x = (6 ± √32) / 2 = 3 ± 2√2
  5. x ≈ 5,83 of x ≈ 0,17. Controleer: x − 2 ≥ 0 → x ≥ 2, dus x ≈ 5,83 = 3 + 2√2
  6. Maar de exacte gehele controle: voor x=7: √17 ≠ 5. Kijk — hier heeft de vergelijking geen geheel getal als oplossing. Exacte antwoord: x = 3 + 2√2 ≈ 5,83. Maar als de vraag een geheel getal vraagt: hercheck de vergelijking. Hier is x=7: √(14+3)=√17≠5. Dus exacte oplossing is 3 + 2√2.
Na kwadreren krijg je een kwadratische vergelijking.
Opgave 12
x + √(7 − x) = 5
x =
  1. Isoleer: √(7−x) = 5 − x
  2. Kwadrateer: 7 − x = 25 − 10x + x²
  3. Herschrijf: x² − 9x + 18 = 0
  4. Ontbind: (x−3)(x−6) = 0 → x=3 of x=6
  5. Controleer x=6: 5−x = −1 < 0, maar √ ≥ 0 ✗ (vreemd)
  6. Controleer x=3: √4 = 2 en 5−3 = 2 ✓ → x = 3
Isoleer de wortelterm eerst.
Opgave 13
√(3x − 2) + 2 = x
x =
  1. Isoleer: √(3x−2) = x − 2
  2. Voorwaarde: x − 2 ≥ 0, dus x ≥ 2
  3. Kwadrateer: 3x − 2 = x² − 4x + 4
  4. Herschrijf: x² − 7x + 6 = 0
  5. Ontbind: (x−1)(x−6) = 0 → x=1 of x=6
  6. x=1 valt buiten domein (x ≥ 2). Controleer x=6: √16 + 2 = 6 ✓ → x = 6
Isoleer √, check of rechterkant ≥ 0.
Opgave 14
2√x + x = 8
x =
  1. Stel u = √x (dus u ≥ 0 en x = u²)
  2. Vergelijking wordt: 2u + u² = 8 → u² + 2u − 8 = 0
  3. Ontbind: (u+4)(u−2) = 0 → u = −4 (ongeldig) of u = 2
  4. Dus √x = 2 → x = 4
  5. Controleer: 2·2 + 4 = 8 ✓
Stel u = √x en los op als kwadratische vergelijking.
Opgave 15
√x + √(x+3) = 3
x =
  1. Isoleer één wortel: √(x+3) = 3 − √x
  2. Kwadrateer: x + 3 = 9 − 6√x + x
  3. Vereenvoudig: 3 = 9 − 6√x → 6√x = 6 → √x = 1
  4. Kwadrateer: x = 1
  5. Controleer: √1 + √4 = 1 + 2 = 3 ✓
