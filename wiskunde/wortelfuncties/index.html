Theorie A — Wortelfuncties & transformaties

Wat is een wortelfunctie?

De eenvoudigste wortelfunctie is:

f(x) = √x

Dit betekent: voor elke x-waarde bereken je de vierkantswortel. Bijvoorbeeld: f(4) = √4 = 2, en f(9) = √9 = 3.

Belangrijk: Je kunt alleen de wortel trekken uit een getal dat ≥ 0 is. Daarom geldt voor f(x) = √x dat x ≥ 0 moet zijn.

De algemene vorm

In de meeste opgaven krijg je een getransformeerde wortelfunctie:

f(x) = a · √(x − p) + q

Wat doen a, p en q? Laten we ze stuk voor stuk bekijken:

Parameter a — vermenigvuldiging & spiegeling

De factor a bepaalt hoe steil de grafiek is en of hij omhoog of omlaag gaat:

  • a > 1: grafiek wordt steiler (verticaal uitgerekt)
  • 0 < a < 1: grafiek wordt vlakker (verticaal samengedrukt)
  • a < 0: grafiek wordt gespiegeld in de x-as (gaat naar beneden)
Voorbeeld: f(x) = 2√x gaat steiler omhoog dan √x.
f(x) = −√x gaat juist naar beneden (gespiegeld).

Parameter p — horizontale verschuiving

De waarde p verschuift de grafiek naar links of rechts:

  • p > 0: grafiek schuift naar rechts
  • p < 0: grafiek schuift naar links
Let op het minteken! In de formule staat (x − p). Als er staat √(x + 3), dan is dat √(x − (−3)), dus p = −3 en de grafiek gaat 3 naar links. Dit is een veelgemaakte fout!

Parameter q — verticale verschuiving

De waarde q verschuift de hele grafiek omhoog of omlaag:

  • q > 0: grafiek schuift omhoog
  • q < 0: grafiek schuift omlaag

Theorie B — Domein & bereik

Domein bepalen

Het domein is de verzameling van alle x-waarden waarvoor de functie bestaat. Bij wortelfuncties moet alles onder het wortelteken ≥ 0 zijn.

f(x) = a · √(x − p) + q → domein: x ≥ p
Voorbeeld 1: f(x) = √(x − 3) → x − 3 ≥ 0 → x ≥ 3

Voorbeeld 2: f(x) = 3 − √(x² − 4)
Onder de wortel: x² − 4 ≥ 0 → x² ≥ 4 → x ≤ −2 of x ≥ 2
(Dit is lastiger! Het is een parabool-ongelijkheid.)
Stappenplan domein:
  1. Schrijf op wat er onder het wortelteken staat
  2. Stel dat ≥ 0
  3. Los de ongelijkheid op

Bereik bepalen

Het bereik is de verzameling van alle y-waarden die de functie aanneemt.

  • Als a > 0: de grafiek stijgt vanaf q → bereik: y ≥ q
  • Als a < 0: de grafiek daalt vanaf q → bereik: y ≤ q
Voorbeeld: f(x) = −2√(x + 4) + 2
Hier is a = −2 (negatief) en q = 2.
De grafiek start op y = 2 en gaat naar beneden → bereik: y ≤ 2

Theorie C — Randpunten

Wat is een randpunt?

Een randpunt is een punt waar de grafiek begint (of eindigt). Bij een wortelfunctie is dat het punt waar de uitdrukking onder de wortel precies 0 wordt.

Randpunt van f(x) = a · √(x − p) + q is het punt (p, q)

Dit is logisch: als x = p, dan staat er √0 = 0, en de functiewaarde is dan a · 0 + q = q.

Voorbeeld: f(x) = 3 − √(x² − 4)
Randpunten vind je waar x² − 4 = 0 → x² = 4 → x = 2 of x = −2.
• x = 2: f(2) = 3 − √0 = 3 → randpunt (2, 3)
• x = −2: f(−2) = 3 − √0 = 3 → randpunt (−2, 3)

Wanneer heeft een grafiek geen randpunten?

Als de uitdrukking onder de wortel altijd ≥ 0 is (nooit precies 0 wordt op een “grens”), dan heeft de grafiek geen randpunt. De grafiek loopt dan eindeloos door.

Voorbeeld: f(x) = 6 − √(x² − 4x + 5)
Bekijk x² − 4x + 5: discriminant = 16 − 20 = −4 < 0.
Omdat de discriminant negatief is en de parabool omhoog opent, is x² − 4x + 5 altijd > 0.
De wortel is dus altijd gedefinieerd en wordt nooit 0 → geen randpunten.

Theorie D — Wortelvergelijkingen

Wat is een wortelvergelijking?

Een wortelvergelijking is een vergelijking met een wortelteken erin, zoals √(2x − 5) = 3.

Stappenplan: wortelvergelijking oplossen

  1. Isoleer de wortel aan één kant van het = teken.
  2. Kwadrateer beide kanten om het wortelteken te laten verdwijnen.
  3. Los de vergelijking op.
  4. Controleer je antwoord! Door het kwadrateren kun je een schijnoplossing krijgen.
Voorbeeld: Los op: √(2x − 5) = 3

Stap 1: Wortel staat al alleen.
Stap 2: Kwadrateer beide kanten: 2x − 5 = 9
Stap 3: 2x = 14, dus x = 7.
Stap 4: Controle: √(2·7 − 5) = √9 = 3. Klopt!
Pas op voor schijnoplossingen!
Neem de vergelijking √(2x − 5) = −3.
Een wortel is altijd ≥ 0, dus √(iets) kan nooit −3 zijn.
Deze vergelijking heeft geen oplossing!
Vuistregel: Als het rechterlid negatief is, heeft de wortelvergelijking geen oplossing. Een wortel geeft altijd een niet-negatief getal.

10 Oefensommen

Test je kennis! Bij elke opgave kun je een hint en de volledige uitwerking bekijken.

1. Wat is het domein van f(x) = √(x − 3)?

Vul het getal in: x ≥ ...

Aanpak: Bij domein van een wortelfunctie stel je altijd de uitdrukking onder het wortelteken ≥ 0.
Hier staat onder de wortel: x − 3. Stel dus op: x − 3 ≥ 0 en los op naar x.
We eisen: x − 3 ≥ 0
Dus: x ≥ 3
Antwoord: x ≥ 3

2. De grafiek van g(x) = √(x + 2) is de grafiek van √x verschoven. Hoeveel en welke richting?

Schrijf bijv. "2 links" of "3 rechts"

Aanpak: Vergelijk met de standaardvorm √(x − p). Herschrijf x + 2 als x − (−2).
Dan is p = −2. Onthoud: negatieve p = naar links, positieve p = naar rechts.
g(x) = √(x + 2) = √(x − (−2))
Dus p = −2. Negatieve p betekent naar links.
Antwoord: 2 naar links

3. Bepaal het startpunt van de grafiek van h(x) = 3√(x − 1) + 5.

Schrijf als (x, y)

Aanpak: Het startpunt (randpunt) van f(x) = a√(x − p) + q is altijd het punt (p, q).
Lees p en q af: wat staat er achter het minteken in (x − ...)? Dat is p. Wat staat er los achter de wortel? Dat is q.
h(x) = 3√(x − 1) + 5, dus a = 3, p = 1, q = 5.
Het startpunt (randpunt) is (p, q) = (1, 5).
Controle: h(1) = 3√0 + 5 = 5. Klopt!
Antwoord: (1, 5)

4. Geef het bereik van f(x) = −2√(x + 4) + 2.

Vul het getal in: y ≤ ... of y ≥ ...

Aanpak: Kijk eerst naar het teken van a. Hier is a = −2 (negatief), dus de grafiek gaat naar beneden.
Bij een dalende wortelfunctie is q het maximum. Het bereik wordt dan y ≤ q. Wat is q hier?
a = −2 < 0, dus de grafiek gaat naar beneden.
Het startpunt is (−4, 2), en de grafiek daalt daarvandaan.
Het maximum is q = 2.
Antwoord: y ≤ 2

5. Gegeven f(x) = 2√(7 − 3x) − 4. Voor welke x geldt f(x) > −2?

Schrijf als ongelijkheid, bijv. "x < 1" (gebruik < of >)

Aanpak: Vul de ongelijkheid in: 2√(7 − 3x) − 4 > −2.
Stap 1: Breng −4 naar rechts → 2√(7 − 3x) > 2.
Stap 2: Deel door 2 → √(7 − 3x) > 1.
Stap 3: Kwadrateer beide kanten (mag, want allebei positief).
Let op: als je deelt door een negatief getal, draait het ongelijkheidsteken om!
f(x) > −2
2√(7 − 3x) − 4 > −2
2√(7 − 3x) > 2
√(7 − 3x) > 1
Kwadrateren (mag, want beide kanten > 0):
7 − 3x > 1
−3x > −6
x < 2 (ongelijkheid draait om bij delen door negatief getal!)
Gecombineerd met domein (7 − 3x ≥ 0 → x ≤ 7/3):
Antwoord: x < 2

6. Gegeven f(x) = 3 − √(x² − 4). Bereken de coördinaten van de twee randpunten.

Schrijf als twee punten, bijv. "(-2, 3) en (2, 3)"

Aanpak: Randpunten vind je waar de uitdrukking onder de wortel precies 0 is.
Stel op: x² − 4 = 0. Los op (ontbinden of wortel trekken).
Vul daarna elke gevonden x in de functie in om de y-waarde te berekenen.
Randpunten: x² − 4 = 0
x² = 4
x = 2 of x = −2

f(2) = 3 − √0 = 3 → punt (2, 3)
f(−2) = 3 − √0 = 3 → punt (−2, 3)
Antwoord: (−2, 3) en (2, 3)

7. Los op: √(2x − 5) = 3

Vul de waarde van x in

Aanpak: Dit is een wortelvergelijking. Gebruik het stappenplan:
1. Wortel staat al alleen aan één kant.
2. Kwadrateer beide kanten: (√(2x − 5))² = 3² → 2x − 5 = 9.
3. Los de gewone vergelijking op.
4. Vergeet niet te controleren door je antwoord terug in te vullen!
√(2x − 5) = 3
Kwadrateer: 2x − 5 = 9
2x = 14
x = 7
Controle: √(14 − 5) = √9 = 3. Klopt!
Antwoord: x = 7

8. Heeft de vergelijking √(2x − 5) = −3 een oplossing? Waarom wel/niet?

Antwoord "ja" of "nee"

Aanpak: Kijk eerst naar het rechterlid van de vergelijking: dat is −3.
Bedenk: een wortelteken √(...) levert altijd een getal ≥ 0 op.
Kan het linkerlid ooit gelijk zijn aan een negatief getal?
Een wortelteken levert altijd een waarde ≥ 0 op.
√(iets) kan dus nooit −3 zijn.
Antwoord: nee, geen oplossing

9. Geef het domein van f(x) = 6 − √(x² − 4x + 5).

Schrijf "alle reele getallen" of een beperking

Aanpak: Je moet weten wanneer x² − 4x + 5 ≥ 0 is.
Bereken de discriminant: D = b² − 4ac = (−4)² − 4·1·5.
Als D < 0 en de parabool opent omhoog (a > 0), dan is de uitdrukking altijd positief.
Wat betekent dat voor het domein?
We moeten kijken: wanneer is x² − 4x + 5 ≥ 0?
Discriminant: D = 16 − 20 = −4 < 0.
De parabool x² − 4x + 5 opent omhoog (positieve x²-term) en heeft geen nulpunten.
Dus x² − 4x + 5 is altijd positief.
De wortel is altijd gedefinieerd.
Antwoord: alle reële getallen

10. Het randpunt van f(x) = 4 − √(5 + ax) ligt op de lijn y = 2x − 1. Bereken exact de waarde van a.

Vul de waarde van a in (als breuk mag ook)

Aanpak: Dit is een puzzel in 3 stappen:
1. Vind het randpunt: stel 5 + ax = 0 → x = −5/a. De y-waarde bij het randpunt is altijd 4 (want √0 = 0).
2. Het randpunt is dus (−5/a, 4).
3. Dit punt ligt op y = 2x − 1. Vul x = −5/a en y = 4 in die lijn in en los op naar a.
Randpunt: 5 + ax = 0 → x = −5/a.
Bij het randpunt is √0 = 0, dus y = 4 − 0 = 4.
Het randpunt is (−5/a, 4).

Dit punt ligt op y = 2x − 1:
4 = 2 · (−5/a) − 1
4 = −10/a − 1
5 = −10/a
a = −10/5 = −2
Antwoord: a = −2
