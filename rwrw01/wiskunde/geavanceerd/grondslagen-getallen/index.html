Waarom dit ertoe doet

Je kunt rekenen. Dat weet je. Maar kun je bewijzen waarom 2 + 3 = 3 + 2? Of waarom “min keer min is plus”?

In dit hoofdstuk pakken we het anders aan. We beginnen bij het allereerste begin: 12 spelregels (wiskundigen noemen ze axioma’s) waaruit álle rekenregels volgen. Een axioma is een spelregel die we als waarheid aannemen — je hoeft het niet te bewijzen, het is het startpunt.

Drie soorten axioma’s:
• Lichaamsaxioma’s (P1–P9) — de rekenregels: hoe optellen en vermenigvuldigen werken.
• Orde-axioma’s (P10–P12) — de rangorde: hoe “groter dan” werkt.
• Later komt er nog een dertiende axioma (completheid) dat de stap maakt van breuken naar alle reële getallen — maar dat is voor een volgend hoofdstuk.

Na dit hoofdstuk weet je niet alleen dat rekenregels werken, maar waarom ze werken — en kun je het bewijzen.

B. Lichaamsaxioma’s — de rekenregels (P1–P9)

Lichaam (Engels: field) klinkt abstract, maar het betekent simpelweg: een verzameling getallen waarmee je kunt optellen, aftrekken, vermenigvuldigen en delen. De volgende 9 spelregels beschrijven hoe dat werkt.

P1: Associativiteit van optelling

a + (b + c) = (a + b) + c

Associativiteit — dat is een moeilijk woord voor iets simpels: het maakt niet uit waar je de haakjes zet bij optellen.

Voorbeeld: 3 + (5 + 7) = 3 + 12 = 15, en (3 + 5) + 7 = 8 + 7 = 15. Zelfde uitkomst!

P2: Identiteitselement van optelling

a + 0 = 0 + a = a

Identiteitselement — het ‘niets-doen-getal’: tel je 0 ergens bij op, dan verandert er niets. Nul is neutraal bij optelling.

Probeer zelf: Welk getal verandert niets bij optelling?

P3: Inverse van optelling

Voor elke a bestaat −a, zodat a + (−a) = 0

Inverse — het ‘tegenovergestelde’: elk getal heeft een tegenhanger waarmee je weer op 0 uitkomt. Het tegenovergestelde van 5 is −5, want 5 + (−5) = 0.

Hieruit volgt: Aftrekken IS optellen van het tegenovergestelde.
a − b = a + (−b). Aftrekken is geen aparte bewerking — het is optelling in vermomming!
Probeer zelf: Wat is het tegenovergestelde van 8?

P4: Commutativiteit van optelling

a + b = b + a

Commutativiteit — de volgorde maakt niet uit bij optellen. Of je nu 3 + 7 of 7 + 3 rekent, je krijgt altijd 10.

P5: Associativiteit van vermenigvuldiging

a · (b · c) = (a · b) · c

Zelfde verhaal als P1, maar dan voor vermenigvuldigen: het maakt niet uit waar je de haakjes zet.

Voorbeeld: 2 · (4 · 5) = 2 · 20 = 40, en (2 · 4) · 5 = 8 · 5 = 40. Zelfde uitkomst!

P6: Identiteitselement van vermenigvuldiging

a · 1 = 1 · a = a

Het ‘niets-doen-getal’ voor vermenigvuldiging is 1. Vermenigvuldig je iets met 1, dan verandert er niets.

Belangrijk: 1 ≠ 0. Dit klinkt vanzelfsprekend, maar het is wiskundig essentieel. Het niets-doen-getal voor optelling (0) is een ánder getal dan het niets-doen-getal voor vermenigvuldiging (1). Als ze gelijk zouden zijn, zou elk getal gelijk zijn aan 0 — en dan heb je geen bruikbare wiskunde meer.

P7: Inverse van vermenigvuldiging

Voor elke a ≠ 0 bestaat a¹ (geschreven a−1 of 1/a), zodat a · a−1 = 1

Multiplicatieve inverse — elk getal (behalve 0) heeft een ‘omgekeerde’ waarmee je weer op 1 uitkomt. Het omgekeerde van 5 is ⅕, want 5 · ⅕ = 1.

Hieruit volgt: Delen IS vermenigvuldigen met het omgekeerde.
a / b = a · b−1. Net als aftrekken optelling in vermomming is, is deling vermenigvuldiging in vermomming!
⚠ Waarom niet delen door 0?
Stel dat 0 wél een inverse had. Dan zou er een getal x moeten bestaan met 0 · x = 1.
Maar we bewijzen straks (bij P9) dat a · 0 = 0 voor elk getal a.
Dus 0 · x = 0, niet 1. Tegenspraak!
Daarom bestaat 0−1 niet. Dit is geen willekeurige regel — het volgt logisch uit de andere axioma’s.
Probeer zelf: Wat is de (multiplicatieve) inverse van 4? Schrijf als breuk.

P8: Commutativiteit van vermenigvuldiging

a · b = b · a

De volgorde maakt niet uit bij vermenigvuldigen. 3 · 7 = 7 · 3 = 21.

P9: Distributiviteit

a · (b + c) = a · b + a · c

Distributiviteit — de brug tussen optellen en vermenigvuldigen. Je mag een vermenigvuldiging ‘uitdelen’ over een optelling. Dit is misschien wel de belangrijkste eigenschap, want het verbindt de twee bewerkingen.

Voorbeeld: 3 · (4 + 6) = 3 · 4 + 3 · 6 = 12 + 18 = 30. Klopt, want 3 · 10 = 30.
Probeer zelf: Werk uit: 5 · (7 + 3) = 5 · 7 + 5 · 3 = + =

Eerste bewijzen met P1–P9

Nu het spannende deel: we gaan dingen bewijzen die je altijd als vanzelfsprekend hebt aangenomen.

Bewijs: a · 0 = 0

Waarom is “iets keer nul altijd nul”? Dat volgt uit P9 (distributiviteit):

1 We weten: 0 + 0 = 0 (want P2: 0 is het identiteitselement)
2 Dus: a · 0 = a · (0 + 0) (we vervangen 0 door 0 + 0)
3 P9 toepassen: a · (0 + 0) = a · 0 + a · 0
4 Nu hebben we: a · 0 = a · 0 + a · 0
5 Trek a · 0 af van beide kanten (P3): 0 = a · 0 ✔
Welk axioma gebruikten we in stap 3?

Bewijs: (−a) · b = −(a · b)

We moeten laten zien dat (−a) · b het tegenovergestelde is van a · b. Oftewel: hun som moet 0 zijn.

1 Bereken: a · b + (−a) · b
2 P9 (andersom): = (a + (−a)) · b
3 P3: a + (−a) = 0, dus = 0 · b
4 Zojuist bewezen: 0 · b = 0 ✔

De som is 0, dus (−a) · b is inderdaad het tegenovergestelde van a · b.

Bewijs: (−a)(−b) = a · b — “Min keer min is plus”

Dit is de beroemde regel die iedereen kent maar bijna niemand kan bewijzen. Hier gaat-ie:

1 We weten nu: (−a) · b = −(a · b) (net bewezen)
2 Vul b = −b in: (−a) · (−b) = −(a · (−b))
3 Maar a · (−b) = −(a · b) (zelfde regel, nu met a en −b)
4 Dus: (−a)(−b) = −(−(a · b)) = a · b ✔

In stap 4 gebruiken we dat het tegenovergestelde van het tegenovergestelde je weer terugbrengt bij het origineel: −(−x) = x. Dat volgt uit P3: als x + (−x) = 0, dan is −x + x = 0, dus x is het tegenovergestelde van −x.

C. Orde-axioma’s — de rangorde (P10–P12)

De eerste 9 axioma’s zeggen hoe je kunt rekenen. Maar ze zeggen niets over welk getal groter is dan welk ander getal. Daarvoor hebben we drie extra spelregels nodig.

We definiëren een speciale verzameling P (van “positief”): de verzameling van alle positieve getallen.

P10: Trichotomie

Voor elk getal a geldt precies één van deze drie: a ∈ P,   a = 0,   −a ∈ P

Trichotomie — drie mogelijkheden, en precies één is waar. Net als bij een wedstrijd: je wint, verliest, of speelt gelijk — nooit twee tegelijk.

Dit betekent: elk getal heeft een vaste plek op de getallenlijn. Een getal kan niet tegelijk positief en negatief zijn, en er zijn geen “gaten” in de ordening.

We schrijven a > b als a − b ∈ P (het verschil is positief).

P11: Gesloten onder optelling

Als a > 0 en b > 0, dan a + b > 0

In gewone taal: positief plus positief is altijd positief. Logisch, toch? Maar het is niet vanzelfsprekend — het is een spelregel die we aannemen.

Gevolg: Als a < b, dan a + c < b + c (voor elk getal c).
Bewijs: b − a > 0 (gegeven), dus (b + c) − (a + c) = b − a > 0. ✔

P12: Gesloten onder vermenigvuldiging

Als a > 0 en b > 0, dan a · b > 0

Positief keer positief is altijd positief.

Gevolg: 1 > 0

Dit is een van de mooiste kleine bewijzen. We weten:

11 ≠ 0 (axioma P6)
2Dus (P10): 1 ∈ P of −1 ∈ P
3Stel 1 ∈ P → dan is 1 > 0 ✔
4Stel −1 ∈ P → dan (P12): (−1) · (−1) ∈ P → 1 ∈ P ✔
5In beide gevallen: 1 > 0!

Gevolg: als a ≠ 0, dan a² > 0

Als a > 0: dan a · a > 0 (P12). Als a < 0: dan −a > 0, dus (−a)(−a) = a² > 0 (P12 + min×min=plus).

Belangrijke afleidingen

Als a < b en c > 0, dan ac < bc
Bewijs: b − a > 0 en c > 0, dus (b − a) · c > 0 (P12), dus bc − ac > 0. ✔
⚠ Pas op! Als a < b en c < 0, dan ac > bc
De ongelijkheid draait om bij vermenigvuldiging met een negatief getal!
Bewijs: b − a > 0 en −c > 0, dus (b − a)(−c) > 0, dus −bc + ac > 0, dus ac > bc. ✔

D. Absolute waarde en driehoeksongelijkheid

Definitie

|a| = a als a ≥ 0,    |a| = −a als a < 0

Absolute waarde — de afstand van een getal tot nul op de getallenlijn. Altijd positief of nul.

Voorbeelden: |5| = 5,   |−3| = 3,   |0| = 0

Stelling 1: De driehoeksongelijkheid

|a + b| ≤ |a| + |b|

De ‘kortste weg’-regel: de directe afstand is nooit langer dan de omweg. Als je eerst a stappen doet en dan b stappen, kan de totale verplaatsing nooit groter zijn dan de som van de afstanden — want stappen in tegengestelde richting heffen elkaar (deels) op.

Voorbeeld: a = 5, b = −3.
|a + b| = |5 + (−3)| = |2| = 2
|a| + |b| = 5 + 3 = 8
Inderdaad: 2 ≤ 8 ✔

Bewijs (gevallenanalyse)

We onderscheiden twee gevallen:

1 Geval 1: a en b hebben hetzelfde teken (beide ≥ 0 of beide ≤ 0).
Dan |a + b| = |a| + |b|. Gelijkheid!
2 Geval 2: a en b hebben tegengestelde tekens.
Dan heffen ze elkaar deels op, dus |a + b| < |a| + |b|. Strikte ongelijkheid!

De slimme truc: je kunt het ook bewijzen via |a|² = a². Omdat a² + 2ab + b² ≤ a² + 2|a||b| + b² (want ab ≤ |a||b|), geldt (a+b)² ≤ (|a|+|b|)², en na worteltrekken volgt |a+b| ≤ |a|+|b|.

Wanneer geldt gelijkheid?

E. Uitdaging — bewijsopgaven

Klaar voor het echte werk? Bij deze opgaven moet je zelf een bewijs opschrijven. Noem welke axioma’s (P1–P12) je gebruikt.

1. Bewijs: als bx = b voor een b ≠ 0, dan x = 1.

Hint: welk axioma garandeert dat b een inverse heeft?

Vermenigvuldig beide kanten met b−1 (die bestaat vanwege P7, want b ≠ 0).
Bewijs:
bx = b   (gegeven)
b−1 · (bx) = b−1 · b   (vermenigvuldig met b−1, bestaat vanwege P7)
(b−1 · b) · x = 1   (P5: associativiteit, P7: b−1 · b = 1)
1 · x = 1   (P7)
x = 1   (P6) ✔

2. Bewijs: p² − q² = (p − q)(p + q)

Werk (p − q)(p + q) uit met distributiviteit (P9). Je krijgt 4 termen, waarvan er 2 tegen elkaar wegvallen.
Bewijs (P9 twee keer toepassen):
(p − q)(p + q)
= p(p + q) + (−q)(p + q)   (P9)
= p² + pq + (−q)p + (−q)q   (P9 nogmaals)
= p² + pq − pq − q²   ((−q)p = −(qp) en (−q)q = −q²)
= p² − q²   (pq − pq = 0, P3) ✔

3. Fout zoeken: het “bewijs” dat 2 = 1. Waar gaat het mis?

Neem a = b. Dan:
a² = ab   (beide kanten × a)
a² − b² = ab − b²   (trek b² af)
(a − b)(a + b) = b(a − b)   (ontbinden)
a + b = b   (delen door (a − b))
b + b = b   (want a = b)
2b = b, dus 2 = 1 ???
Kijk naar de stap waar we “delen”. Waardoor delen we? En wat is de waarde daarvan?
De fout zit in stap 4: we delen door (a − b).
Maar a = b, dus a − b = 0.
Delen door 0 is niet toegestaan! (P7: alleen getallen ≠ 0 hebben een inverse.)
Dit is precies waarom de warning-box bij P7 zo belangrijk is: zonder die beperking kun je elke onzin “bewijzen”.

4. Bewijs: ab = acbc als b, c ≠ 0

Schrijf a/b als a · b−1 en ac/(bc) als (ac) · (bc)−1. Gebruik dat (bc)−1 = b−1 · c−1.
Bewijs:
ac/(bc) = (ac) · (bc)−1   (definitie deling)
= (ac) · (b−1 · c−1)   (inverse van product)
= a · (c · c−1) · b−1   (P5, P8: herschikken)
= a · 1 · b−1   (P7: c · c−1 = 1)
= a · b−1   (P6)
= a/b ✔

5. Bewijs: ab + cd = ad + bcbd

Schrijf a/b als a · b−1 en c/d als c · d−1. Maak dan de noemers gelijk door slim te vermenigvuldigen.
Bewijs:
a/b + c/d
= a · b−1 + c · d−1   (definitie)
= a · b−1 · (d · d−1) + c · d−1 · (b · b−1)   (P7: x · x−1 = 1)
= (ad) · (bd)−1 + (cb) · (db)−1   (herschikken)
= (ad) · (bd)−1 + (bc) · (bd)−1   (P8)
= (ad + bc) · (bd)−1   (P9: uitbuiken)
= (ad + bc) / (bd) ✔

Dit is de kruislings-methode die je al kent van breuken optellen!

6. Vind alle x waarvoor 3 − x < 5 − 2x

Tel 2x op bij beide kanten, en trek 3 af van beide kanten.
3 − x < 5 − 2x
3 − x + 2x < 5 − 2x + 2x   (P11-gevolg: tel 2x op)
3 + x < 5
x < 2   (trek 3 af) ✔

7. Vind alle x waarvoor 7 − x² < 3

Herschrijf als x² > 4, en gebruik dan opgave 2 (verschil van kwadraten).
7 − x² < 3
−x² < −4   (trek 7 af)
x² > 4   (vermenigvuldig met −1, ongelijkheid draait om!)
x² − 4 > 0
(x − 2)(x + 2) > 0   (opgave 2!)

Dit geldt als beide factoren positief OF beide negatief:
• x > 2 en x > −2 → x > 2
• x < 2 en x < −2 → x < −2

Antwoord: x < −2 of x > 2 ✔

8. Bewijs: als a < b, dan −b < −a

Tel −a en −b op bij beide kanten van a < b.
Bewijs:
a < b   (gegeven)
a + (−a) + (−b) < b + (−a) + (−b)   (P11-gevolg: tel −a − b op)
−b < −a   (P2, P3: vereenvoudigen) ✔

9. Bewijs: als 0 < a < 1, dan a² < a

a² = a · a. Vermenigvuldig de ongelijkheid a < 1 met a (en a > 0!).
Bewijs:
a < 1   (gegeven)
a · a < 1 · a   (vermenigvuldig met a; mag want a > 0, P12-gevolg)
a² < a   (P6: 1 · a = a) ✔

10. Schrijf |√3 + √5 − √2 − √7| zonder absolute-waardetekens.

Tip: bereken eerst of de uitdrukking positief of negatief is.

√3 ≈ 1,73 en √5 ≈ 2,24 en √2 ≈ 1,41 en √7 ≈ 2,65. Tel op en trek af.
√3 + √5 ≈ 1,73 + 2,24 = 3,97
√2 + √7 ≈ 1,41 + 2,65 = 4,06
Dus √3 + √5 − √2 − √7 ≈ −0,09 < 0

Omdat de uitdrukking negatief is:
|√3 + √5 − √2 − √7| = √2 + √7 − √3 − √5 ✔

11. Vind alle x waarvoor |x − 4| < 3

|x − 4| < 3 betekent: de afstand van x tot 4 is kleiner dan 3. Welke getallen liggen minder dan 3 van 4 af?
|x − 4| < 3 betekent:
−3 < x − 4 < 3
−3 + 4 < x < 3 + 4   (tel 4 op)
1 < x < 7 ✔

12. Bewijs: |a + b| = |a| + |b| als a en b hetzelfde teken hebben.

Behandel twee gevallen: (1) a ≥ 0 en b ≥ 0, en (2) a ≤ 0 en b ≤ 0.
Bewijs (twee gevallen):

Geval 1: a ≥ 0 en b ≥ 0
• a + b ≥ 0 (P11), dus |a + b| = a + b
• |a| = a en |b| = b
• Dus |a + b| = a + b = |a| + |b| ✔

Geval 2: a ≤ 0 en b ≤ 0
• Dan −a ≥ 0 en −b ≥ 0 (P10)
• a + b ≤ 0 (want beide ≤ 0), dus |a + b| = −(a + b) = (−a) + (−b)
• |a| = −a en |b| = −b
• Dus |a + b| = (−a) + (−b) = |a| + |b| ✔
