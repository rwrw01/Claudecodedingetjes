1. Rekenregels voor machten (herhaling)

Voordat we negatieve en gebroken exponenten behandelen, herhalen we de basisregels.

Regel Formule Voorbeeld
Vermenigvuldigen (zelfde grondtal) ap · aq = ap+q y3 · y5 = y8
Delen (zelfde grondtal) apaq = ap−q z7z3 = z4
Macht van een macht (ap)q = ap·q (x2)4 = x8
Macht van een product (ab)p = apbp (3x)3 = 27x3
💡 Tip: Deze regels gelden voor elk getal als exponent — ook voor negatieve en gebroken exponenten!
2² · 2³ = 4 · 8 = 32 = 2⁵ ✓

🔸 Snelle herhaling

Bereken: x4 · x6 = x?
Tel de exponenten op: 4 + 6 = ?
Uitwerking: x4 · x6 = x4+6 = x10
Vereenvoudig: a8a3 = a?
Trek de exponenten van elkaar af: 8 − 3 = ?
Uitwerking: a8 / a3 = a8−3 = a5
Vereenvoudig: (y3)4 = y?
Vermenigvuldig de exponenten: 3 × 4 = ?
Uitwerking: (y3)4 = y3×4 = y12

2. De nulde macht: a⁰ = 1

Stel: a ≠ 0. Wat is a0?

🧠 Redeneer zelf

Gebruik de deelregel: apaq = ap−q

Stap 1: vul p = q = 5 in:

a5a5 = a5−5 = a0

Stap 2: maar a5a5 is hetzelfde als...

a5a5 = 1   (want teller = noemer)

Conclusie:

Dus a0 = 1 voor alle a ≠ 0. ✓
a0 = 1   (voor a ≠ 0)
⚠️ Let op: 00 is ongedefinieerd! De regel geldt alleen als a ≠ 0.

3. Theorie A: Negatieve exponenten

Nu gaan we verder: wat betekent een negatieve exponent?

🧠 Afleiding stap voor stap

We weten dat a0 = 1 en apaq = ap−q

a0an = a0−n = a−n
1an = a−n
a−n = 1an
Een negatieve exponent betekent: "zet het getal in de noemer!"
a−q = 1aq   (voor a ≠ 0)
📝 Voorbeelden:
  • x−3 = 1x3
  • 4x−2 = 4 · 1x2 = 4x2
  • 1y5 = y−5
  • 6x = 6x−1
⚠️ Veelgemaakte fout: 4x−2 ≠ (4x)−2
• 4x−2 = 4x2 (alleen x wordt in de noemer gezet)
• (4x)−2 = 1(4x)2 = 116x2 (heel de haakjesterm gaat in de noemer)

Schrijven zonder negatieve exponenten

Je kunt een uitdrukking met negatieve exponenten omschrijven zodat er alleen positieve exponenten overblijven.

📝 Voorbeeld: schrijf 3x−4y2 zonder negatieve exponenten
3x−4y2 = 3 · 1x4 · y2 = 3y2x4
📝 Voorbeeld: schrijf (2x)−3y4 zonder negatieve exponenten
(2x)−3y4 = y4(2x)3 = y48x3
📝 Voorbeeld: schrijf 23x−1 zonder negatieve exponenten
23x−1 = 23 · 1x = 23x

5. Theorie B: Gebroken exponenten

Nu bekijken we wat een gebroken exponent betekent, zoals x1/2 of x3/4.

🧠 Afleiding van x1/2

We gebruiken de regel (ap)q = apq.

(x½)2 = x½ · 2 = x1 = x
Als (x½)² = x, dan is x½ de wortel van x.
Want: (√x)² = x
x½ = √x   ✓
a1q = q√a    en    apq = q√(ap)
📝 Voorbeelden:
  • x1/3 = ³√x
  • x2/3 = ³√(x²)
  • x3/4 = ⁴√(x³)
  • x−1/2 = 1x1/2 = 1√x
💡 Tip: Een gebroken exponent pq kun je lezen als:
• q = de index van de wortel (q-de-machtswortel)
• p = de macht van het grondtal onder de wortel

Schrijven als macht van x

📝 Voorbeeld: schrijf x · ⁴√(x³) als macht van x
Stap 1: ⁴√(x³) = x3/4
Stap 2: x · x3/4 = x1 · x3/4 = x1 + 3/4 = x7/4
📝 Voorbeeld: schrijf x3⁵√x als macht van x
Stap 1: ⁵√x = x1/5
Stap 2: x3x1/5 = x3 − 1/5 = x15/5 − 1/5 = x14/5
📝 Voorbeeld: schrijf ⁴√x⁶√x als macht van x
Stap 1: ⁴√x = x1/4 en ⁶√x = x1/6
Stap 2: x1/4x1/6 = x1/4 − 1/6
Stap 3: gelijknamig maken: x3/12 − 2/12 = x1/12
