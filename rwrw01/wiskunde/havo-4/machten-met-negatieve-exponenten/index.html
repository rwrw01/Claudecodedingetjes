1. Rekenregels voor machten (herhaling)

Voordat we beginnen met negatieve en gebroken exponenten, herhalen we de vier basisregels voor machten.

Regel Notatie Uitleg
Vermenigvuldigen ap · aq = ap+q Zelfde grondtal → exponenten optellen
Delen ap / aq = ap−q Zelfde grondtal → exponenten aftrekken
Macht van een macht (ap)q = ap·q Exponenten vermenigvuldigen
Macht van een product (ab)p = apbp Macht over elk factor verdelen
Voorbeelden:
y3 · y5 = y8  |  z7 / z3 = z4  |  (b4)3 = b12  |  (2y)3 = 8y3

2. Theorie A: Negatieve exponenten

Hoe ontstaat een negatieve exponent?

We passen de deelregel toe: ap / aq = ap−q. Wat als p < q?

Manier 1 (vereenvoudigen): a³ / a³ = 1 (want teller = noemer)
Manier 2 (deelregel): a³ / a³ = a3−3 = a0
➜ Dus: a0 = 1 (voor a ≠ 0)
Manier 1 (inkorten): a²a⁵ = 1a³
Manier 2 (deelregel): a² / a⁵ = a2−5 = a−3
➜ Dus: a−3 = 1/a³
Voor a ≠ 0:
a0 = 1     en     a−q = 1aq
💡 Tip: Een negatieve exponent betekent: "zet het onder de streep". Dus x−4 = 1/x4. De exponent zelf wordt positief als je de breuk schrijft.
⚠️ Veelgemaakte fout: −x4 is NIET hetzelfde als x−4! Het minteken bij de exponent geeft een breuk; een minteken vóór de macht geeft een negatieve waarde.

Voorbeelden: schrijven zonder negatieve exponent

Schrijf zonder negatieve exponenten:

a) 3y−4z2
3y−4z2 = 3z2y4   (de y met negatieve exponent gaat naar de noemer)

b) (3y)−4z2
(3y)−4z2 = z2(3y)4 = z281y4   (let op: de haakjes horen bij de macht!)

c) 25 x−1
25 x−1 = 25 · 1x = 25x

d) (23x)−1
(²⁄₃x)−1 = (²⁄₃)−1 · x−1 = ³⁄₂ · 1x = 32x   (de breuk omdraaien geeft de inverse)

4. Theorie B: Gebroken exponenten

We gebruiken de regel (ap)q = ap·q om de betekenis van x1/n te ontdekken.

Stel p = ½ en q = 2. Dan geldt:
(x1/2)2 = x(1/2)·2 = x1 = x
Maar ook: (√x)2 = x
➜ Dus x1/2 = √x
Stel p = ⅓ en q = 3. Dan geldt:
(x1/3)3 = x1 = x
Maar ook: (∛x)3 = x
➜ Dus x1/3 = ∛x
xp/q = (xp)1/q = q√(xp)
Of ook: xp/q = (x1/q)p = (q√x)p
a1/q = q√a    en    ap/q = q√(ap)
💡 Tip: De noemer van de exponentbreuk geeft de graad van de wortel. Dus bij x2/5: je neemt de 5e-machtswortel van x², geschreven als ⁵√(x²).

Voorbeelden: machten als wortel schrijven

Machtnotatie Wortelnotatie
n1/4⁴√n
y3/4⁴√(y³)
x2/5⁵√(x²)
z−1/31/∛z

Voorbeeld: Schrijf als macht van x

a) x · ∛(x²)
x · ∛(x²) = x1 · x2/3 = x1 + 2/3 = x5/3

b) x³⁴√x
x³⁴√x = x³x1/4 = x3 − 1/4 = x11/4

c) ⁵√x∛x
⁵√x∛x = x1/5x1/3 = x1/5 − 1/3 = x3/15 − 5/15 = x−2/15

🔢 Verken: gebroken exponenten van 8

81/3 = ∛8 = 2. Verken andere breuken als exponent.



81/3 ≈ 2
