De basisvorm

De standaard wortelfunctie is:

f(x) = √x

De grafiek begint in de oorsprong (0, 0) en loopt stijgend naar rechts. Het domein is x ≥ 0 en het bereik is y ≥ 0.

De algemene vorm met transformaties is:

f(x) = a · √(x − p) + q

a = verticale vermenigvuldiging (en spiegeling als a < 0)
p = horizontale verschuiving (naar rechts als p > 0)
q = verticale verschuiving (omhoog als q > 0)

Stapsgewijze opbouw van f(x) = −2√(x + 4) + 2

We herschrijven: f(x) = −2√(x − (−4)) + 2, dus a = −2, p = −4, q = 2. Hieronder bouwen we de grafiek stap voor stap op.

Oefenopgaven

Test je kennis! Vul je antwoord in en controleer.

1. Wat is het domein van f(x) = √(x − 3)?

Schrijf als: x ≥ ... (vul het getal in)

Onder het wortelteken moet ≥ 0 staan: x − 3 ≥ 0.

2. De grafiek van g(x) = √(x + 2) is de grafiek van √x verschoven met ... eenheden naar ...

Antwoord: een getal en richting, bijv. "2 links"

x + 2 = x − (−2), dus p = −2. Negatieve p = naar links.

3. Bepaal het startpunt (beginpunt van de grafiek) van h(x) = 3√(x − 1) + 5.

Schrijf als: (x, y), bijv. (1, 5)

Het startpunt is (p, q). Hier: p = 1, q = 5.

4. Is de grafiek van f(x) = −√x stijgend of dalend?

De min-teken voor de wortel spiegelt de grafiek in de x-as.

5. Geef het bereik van f(x) = −2√(x + 4) + 2.

Schrijf als: y ≤ ... (vul het getal in)

a is negatief, dus de grafiek gaat naar beneden. Het maximum is q = 2.
